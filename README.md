# The new flow
- Landing page 1
- Login/Create account (kan bara vara username/password for now)
- Landing page 2 
    - New game
        - User is prompted to choose:
            - Name
            - Starting location
            - What they are doing
            - One thing in their inventory
        - Game loop is started
    - Continue game
        - Previous game data is fetched from db
        - Game loop is started
